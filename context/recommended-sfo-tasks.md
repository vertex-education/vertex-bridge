# Recommended SFO Demo Tasks

Use a dedicated Asana demo project named:

Heritage Summit Schools - SFO Onboarding

Every task in the project is client-facing and should appear in the School Onboarding portal.

## Recommended Five Tasks

1. Upload 501c3 letter or EIN documentation.
   - Completion type: file upload.
   - Helpful note: If no 501c3 letter is available, EIN/business-name documentation may be used for the demo.

2. Upload voided checks for all bank accounts.
   - Completion type: file upload.
   - Helpful note: If no check stock is available, a bank letter with routing and account numbers may be referenced in task instructions.

3. Upload latest payroll register with YTD info.
   - Completion type: file upload.

4. Upload board-approved current year budget in Excel.
   - Completion type: file upload.

5. Upload preliminary deposits/expenses, General Ledger detail, and Trial Balance Report.
   - Completion type: file upload.
   - This is a good AI-helper step because clients may not know what the accounting terms mean.

## Display Rules

- Show task name as the portal step title.
- Show Asana task notes as client instructions.
- Show Asana due date as the step due date.
- Show Asana completion status as the step status.
- Sort all steps by due date.
- Tasks without a due date should appear at the bottom with "No due date set."
