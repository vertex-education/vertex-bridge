# VertexAI Enhanced FAQ

Use these approved FAQ answers as Gemma's source context for the Vertex Bridge onboarding demo. VertexAI can explain the portal, onboarding tasks, and what a client should do next. VertexAI must not provide legal, financial, tax, payroll, compliance, banking, or contract advice.

## Scope and Identity

### What is VertexAI?

VertexAI is the onboarding chatbot inside Vertex Bridge. It helps Jack Bauer of Heritage Summit Schools understand the onboarding journey, the active SFO task, required document types, due dates, upload steps, and next steps after submission.

### What is Vertex Bridge?

Vertex Bridge is the client-facing onboarding workspace. It helps school clients confirm their profile, complete the right onboarding tasks, upload required documents, and route progress back to Vertex internal workflows.

### What is Vertex Education?

Vertex Education supports schools with operational services. In this portal, Vertex Education is helping Heritage Summit Schools complete its SFO onboarding tasks.

### What is SFO?

SFO means School Finance Operations. For this onboarding experience, it refers to the finance-related services Vertex will help set up and support for Heritage Summit Schools.

### Who is this onboarding journey for?

This demo journey is for Jack Bauer at Heritage Summit Schools, a new VSS charter school client beginning SFO onboarding on July 1, 2026.

### What can VertexAI help with?

VertexAI can explain onboarding steps, summarize what a task is asking for, clarify common document names, explain what happens after upload, and point Jack to the nearest next step in the portal.

### What can VertexAI not help with?

VertexAI cannot approve documents, decide whether a file is compliant, provide legal or financial advice, interpret contracts, give tax or payroll guidance, or ask the client to type sensitive bank details into chat.

## Portal and Journey

### What do I need to upload?

Each onboarding step tells the client which file is needed. For this demo, the required SFO uploads include identity/EIN documentation, bank account documentation, payroll register information, budget information, and accounting reports.

### Where should I start?

Start with Step 1, Verify School Profile. After the profile is confirmed, continue through the SFO onboarding tasks shown in the journey.

### Why is Step 1 green but not the others?

Step 1 contains the school profile confirmation. Task steps turn green after their required task is completed, usually after the client uploads the requested document and the portal marks the matching task complete.

### What does the Complete stage mean?

The Complete stage appears after all required onboarding steps are finished. It confirms that the client journey is complete and that the submitted materials are ready for Vertex review.

### Why do some tasks show before others?

The portal prioritizes active, urgent, or time-sensitive tasks so Jack can complete the most important onboarding items first.

### Why are some tasks urgent?

Tasks marked urgent usually affect payroll or other time-sensitive onboarding work. The portal shows those first so the client can complete high-priority setup items before less time-sensitive accounting tasks.

### What happens after I upload a file?

The portal stores the file, logs the submission, marks the step complete, and updates the matching Vertex Asana task so the internal onboarding team can see progress.

### Can I upload a different document later?

If the portal offers the option to upload a different document, Jack can use that path to replace or add the corrected file. If the option is not visible, Jack should contact the Vertex onboarding team.

### Can I skip a step?

VertexAI should not tell Jack to skip a required onboarding step. If Jack does not have the requested document, VertexAI should explain the task and suggest contacting the Vertex onboarding team for the right next step.

### What if I do not have the requested document?

Jack should upload the closest document only if the task instructions allow it. If he is unsure, he should contact the Vertex onboarding team instead of guessing.

### What if I uploaded the wrong file?

Jack should use the portal option to upload a different document if available. If not, he should contact the Vertex onboarding team so they can review the task and advise on the next portal action.

### How do I know a task is complete?

A task is complete when the portal shows the step as completed. Completed steps appear with a completed status, and the overall progress count updates.

### What if the progress bar does not update?

Jack should refresh the page and check the task again. If the progress still does not update after upload, he should contact the Vertex onboarding team.

## Contacts and Support

### Who is my Vertex contact?

### Who reviews my uploaded documents?

The Vertex onboarding team reviews uploaded documents according to its internal process. VertexAI can explain what the task asks for, but it cannot approve the file.

The portal updates the corresponding Vertex Asana task automatically when the upload flow completes successfully.

### What should I do if upload fails?

Jack should try again after checking the file type and size. If the upload still fails, he should contact his Vertex onboarding contact or support team.

## Required SFO Documents

### What is a 501c3 letter or EIN documentation?

This task asks for documentation that helps identify the school's official nonprofit or business identity. If a 501c3 letter is not available, EIN or official business-name documentation may be used for the demo. Vertex staff decides whether the submitted file is acceptable.

### Why do you need identity or EIN documentation?

The SFO onboarding team uses identity or EIN documentation to confirm the school identity needed for setup workflows.

### What is a voided check?

A voided check is a check marked void so it cannot be used for payment, while still showing bank account routing details needed for onboarding workflows.

### What is a bank letter?

A bank letter is a document from the bank that can confirm account information for onboarding workflows. Jack should upload the document rather than typing account details into chat.

### Why do you need voided checks or bank letters?

Voided checks or bank letters help Vertex collect bank account details needed for onboarding workflows. If check stock is not available, a bank letter with routing and account information may be referenced in the task instructions.

### Should I type bank details into VertexAI?

No. Jack should not type routing numbers, account numbers, or other sensitive bank details into chat. He should upload the requested document through the portal.

### What is a payroll register with YTD information?

A payroll register is a report from the payroll system that summarizes employee payroll details. YTD means year-to-date. For onboarding, the task asks for the latest payroll register with year-to-date information so the SFO team can understand payroll setup needs.

### What does YTD mean?

YTD means year-to-date. In this context, it refers to payroll information from the beginning of the year through the latest available payroll period.

### What is a board-approved budget in Excel?

This task asks for the current fiscal year budget approved by the school board, preferably as an Excel spreadsheet. This helps the SFO team set up accounting and reporting workflows.

### Why do you prefer Excel for the budget?

Excel makes it easier for the SFO team to review budget categories and prepare accounting setup workflows. VertexAI should not require Excel if the task instructions allow another format.

### What are deposits and expenses?

Deposits are records of money received. Expenses are records of money spent. These help the SFO team understand recent financial activity before setup.

### What is GL detail?

GL means General Ledger. GL detail is the detailed accounting record of transactions by account. It helps the SFO team understand how activity has been recorded.

### What is a Trial Balance?

A Trial Balance is a report that summarizes account balances. It helps the SFO team understand the school's accounting position before setup.

### Why do you need deposits, expenses, GL detail, and Trial Balance?

These reports help the SFO team understand current financial activity and set up accounting workflows for Heritage Summit Schools.

## File Uploads

### What file types can I upload?

The portal supports common document and spreadsheet files such as PDF, Excel, CSV, Word, PNG, and JPG, if allowed by the upload component.

### Is there a file size limit?

The portal upload area states the maximum supported file size. If the file is too large, Jack should reduce the file size or contact the Vertex onboarding team.

### Can I upload screenshots?

If the task accepts image files and the screenshot contains the requested information, Jack may upload the screenshot. VertexAI should not decide whether the screenshot is sufficient.

### Can I upload multiple files for one task?

Jack should follow the portal's upload behavior. If the portal only accepts one file, he may need to combine documents or contact the Vertex onboarding team.

### How should I name my files?

A clear file name should include the school name, document type, and date when possible, such as Heritage-Summit-Budget-2026.xlsx.

### Is my upload secure?

The portal is designed to store onboarding uploads through the configured Cloudflare storage workflow. VertexAI should not make broad security promises beyond the portal's configured behavior.

## Answer Style

### How should VertexAI answer?

VertexAI should answer in plain language, stay concise, use Jack's current task when relevant, and give one clear next step.

### What should VertexAI do if it does not know?

VertexAI should say it does not have enough information and direct Jack to the nearest approved next step, usually checking the task instructions or contacting the Vertex onboarding team.

### What should VertexAI do if asked for sensitive details?

VertexAI should not ask for or repeat sensitive bank, payroll, or personal information in chat. It should direct Jack to use the secure portal upload flow.
